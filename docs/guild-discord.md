# Base Guild & Discord роли — пошагово

## 1. Присоединяйся к Guild
1. Перейди → [guild.xyz/base](https://guild.xyz/base) или [era.guild.xyz/base](https://era.guild.xyz/base)
2. Подключи кошелёк
3. Выполни задания:
   - Зарегистрируй Basename
   - Сделай bridge
   - Подключи GitHub (аккаунт старше июля 2025 + хотя бы 1 commit)
   - Посети [base.org/build](https://base.org/build)

## 2. Роли в Discord
1. Зайди в официальный [Discord Base](https://discord.gg/base)
2. Подключи wallet через Guild-бот
3. Получи роли:
   - **Base Initiate**
   - **Base Builder** (нужно 5+ деплоев контрактов)
   - **Builders & Founders** (GitHub + активности)

## 3. Quests и credentials
- Выполняй задания в Guild → получай on-chain proofs
- Чем больше заданий — тем выше статус и rewards

**Проверь роли**: guild.xyz/base после выполнения.
